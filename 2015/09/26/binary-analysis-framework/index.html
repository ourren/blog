Angr:二进制分析框架入门

Sep 26th, 2015 12:05 am | 留言

angr 号称作为新一代二进制分析框架，侧重于静态和动态符号执行。Angr将一些常见的程序分析功能进行了分类和实现，让使用者可以通过Python编程对程序的分析，按照官方文档的介绍主要有如下功能：

  • Disassembly and intermediate-representation lifting
  • Program instrumentation
  • Symbolic execution
  • Control-flow analysis
  • Data-dependency analysis
  • Value-set analysis (VSA)

其中Angr多次帮助shellphish战队在各大CTF取得优势成绩，同时依靠其框架团队在defcon 23，blackhat 2015和hitcon 都有议题呈现，详细议题文档如下。

  • DEFCON-23-Angry-Hacking.pdf;
  • Using Static Binary Analysis to Find Vulnerabilities and Backdoors in Firmware;
  • A Dozen Years of Shellphish - from DEFCON to the Cyber Grand Challenge;

鉴于本人对符号执行这些知识的欠缺，本文主要分享angr的搭建和初步分析，作为个人的学习笔记。

安装

推荐安装环境：Ubuntu + virtualenvwrapper「管理多个不同的Virtualenv虚拟环境」 进行程序分析。

依赖库安装：

sudo apt-get install python-dev libffi-dev build-essential virtualenvwrapper

相关配置：

其中virtualenvwrapper的安装还需要进行相关配置，官方的文档 稍微有些出入，以下是前期设置和一些常见命令。

  export WORKON_HOME=~/sites
  # post-install 
  source /usr/local/bin/virtualenvwrapper.sh
  on Ubuntu 14.04 LTS, the path is 
  source /usr/share/virtualenvwrapper/virtualenvwrapper.sh

  mkvirtualenv angr

  # list all available env
  workon

  # switch to an angr
  workon angr

  setvirtualenvproject # set project for current dir

  cdvirtualenv  # go to virtualenv folder
  lssitepackages # list current installed packages
  cdsitepackages # cd site-packages folder directly

  workon env1
  echo $VIRTUAL_ENV

  deactivate # exit current env

安装angr:

主要分为两种模式，发布模式：

mkvirtualenv angr
pip install angr

或者开发模式：

git clone https://github.com/angr/angr-dev
cd angr-dev
mkvirtualenv angr
./setup.sh

如果安装问题可以看这里 angr.
